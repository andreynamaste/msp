🎯 КАК ТЕПЕРЬ РАБОТАЕТ ПРИЛОЖЕНИЕ
═══════════════════════════════════════════════════════════════════

ВАРИАНТ 1: ВЫБОР ГОРОДА
1. Выберите страну (например: Украина)
   → Часовой пояс автоматически установится (UTC+2)
2. Выберите город (например: Донецк)
   → Часовой пояс обновится на UTC+4 (Донецк)
3. Заполните остальные поля
4. Нажмите "🔮 Рассчитать натальную карту"

ВАРИАНТ 2: ВВОД КООРДИНАТ
1. Нажмите "Ввести координаты"
2. Введите широту и долготу
   → Часовой пояс рассчитается автоматически
3. Заполните остальные поля
4. Нажмите "🔮 Рассчитать натальную карту"

ВАРИАНТ 3: РУЧНОЙ ВВОД ЧАСОВОГО ПОЯСА
1. Введите часы (от -12 до +14)
2. Введите минуты (0-59)
3. Нажмите кнопку "Применить"
   → Часовой пояс установится вручную
4. Заполните остальные поля
5. Нажмите "🔮 Рассчитать натальную карту"
