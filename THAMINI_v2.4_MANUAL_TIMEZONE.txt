═══════════════════════════════════════════════════════════════════
📊 КАК ИСПОЛЬЗОВАТЬ РУЧНОЙ ВВОД ЧАСОВОГО ПОЯСА
═══════════════════════════════════════════════════════════════════

1. Найдите секцию "🕐 Ручной ввод часового пояса (UTC offset)"
2. Введите часы в поле "±часы" (например: +3, -5, +5)
3. Введите минуты в поле "минуты" (например: 0, 30, 45)
4. Нажмите кнопку "Применить"
5. Часовой пояс будет установлен и отображен в выпадающем списке

ПРИМЕРЫ:
• Москва: UTC+3:00 → Часы: 3, Минуты: 0
• Киев: UTC+2:00 → Часы: 2, Минуты: 0
• Индия: UTC+5:30 → Часы: 5, Минуты: 30
• Нью-Йорк: UTC-5:00 → Часы: -5, Минуты: 0
• Лондон: UTC+0:00 → Часы: 0, Минуты: 0

🔬 ТЕХНИЧЕСКИЕ ДЕТАЛИ
═══════════════════════════════════════════════════════════════════

Функция: applyManualTimezone()
• Получает часы и минуты из полей ввода
• Проверяет диапазоны (-12 до +14 часов, 0-59 минут)
• Форматирует как UTC±HH:MM
• Устанавливает в выпадающий список
• Показывает уведомление
